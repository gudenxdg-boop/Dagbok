# Dagbok – kodminne (v0.1)

En superliten digital dagbok som sparar per dag i webbläsaren.

- Visar dagens datum
- Humör-slider (0–100)
- “Random tanke” (eller skriv egen)
- Sparar per dag (localStorage)
- Historik + export/import JSON

Byggd som ett kodminne – inte som en produkt.

## Kör
Öppna `index.html` i webbläsaren.
